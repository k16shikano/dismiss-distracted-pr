# dismiss-distracted-pr

## 概要

Twitter（X）で気が散るツイートを自動で非表示にするChrome拡張機能。

## 特徴
- **プロモーションツイートの自動ミュート**: 「絵文字がある」「ハッシュタグがある」「空行がある」という条件のうち2つ以上を満たすプロモーションツイートのアカウントを自動でミュート
- **フォローしていないアカウントの自動非表示**: フォローしていないアカウントによるツイートを自動で「興味がない」に分類して非表示
- **リツイートの自動フィルタリング**: フォローしている人からのリツイートではないツイートを自動で「興味がない」に分類して非表示
- **自動通知閉じ**: ミュート時のダイアログや通知も自動で閉じる
- **TypeScript実装**: 型安全なコードで実装

## 使い方
1. このリポジトリをクローン
2. `npm install` で依存パッケージをインストール
3. `npm run build` でビルド
4. Chromeの「拡張機能」画面で「パッケージ化されていない拡張機能を読み込む」から`dist`フォルダを選択
5. X（Twitter）のページを開いて使用開始

## ビルド方法
```sh
npm install
npm run build
```
